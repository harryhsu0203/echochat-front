步驟 1: 建立 LINE Messaging API 頻道

建立頻道說明
  1. 前往 LINE Developers Console
  2. 登入您的 LINE 帳號
  3. 點擊「Create」→「Create a new provider」
  4. 輸入 Provider 名稱（例如：您的公司名稱）
  5. 選擇「Messaging API」
  6. 填寫頻道資訊：
    • Channel name: 您的 Bot 名稱
    • Channel description: Bot 功能描述
    • Category: 選擇「Business」
    • Subcategory: 選擇「Customer service」
  7. 同意服務條款並點擊「Create」

步驟 2: 取得頻道資訊

取得必要的資訊
  1. 在您的頻道頁面中，點擊「Messaging API」標籤
  2. 找到「Channel access token」區塊，點擊「Issue」
  3. 複製「Channel access token」
  4. 在「Basic settings」標籤中，找到「Channel secret」
  5. 複製「Channel secret」
Token 格式通常以 "Bearer " 開頭，請完整複製
Secret 是一串英數字組合，請完整複製

步驟 3: 設定 Webhook URL

設定 Webhook URL
  1. 在頻道的「Messaging API」標籤中，找到「Webhook settings」
  2. 點擊「Edit」
  3. 將 Webhook URL 設定為下方提供的網址
  4. 點擊「Update」
  5. 點擊「Verify」驗證 Webhook 是否正常
  6. 將「Use webhook」開關設為「Enabled」
請完整複製此 URL 到 LINE Developers Console
待驗證

步驟 4: 完成設定

LINE Bot 設定完成！

您的 LINE Bot 現在已經準備好提供智能客服服務

  3. 測試發送訊息給您的 LINE Bot
  4. 在知識庫中添加常見問題和回答
